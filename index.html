🎮 Bienvenue dans le Jeu des Défis 🎮

Voici les règles du jeu :

  • 1. Après avoir renseigné les participants, tu peux cliquer sur "C'est parti !" pour démarrer le jeu.
  • 2. Un défi s'affiche pour chaque joueur.
  • 3. Si un participant refuse un défi, clique sur son nom pour lui enlever un point.
  • 4. Une fois tous les défis terminés, consulte le tableau des scores !

Bonne chance et amusez-vous bien ! 😎
